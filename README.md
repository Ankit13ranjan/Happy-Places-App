# Happy-Places-App

A Happy Place App Made Up By Using Android.

Which can Store Memorable Moments with their Feature.

It Have Access to Camera,

It Have Access to Google API For Current Location,

It Uses Recycler View for Storing Activity one by one.

It Have Floating Button Feature For Creating New Record.
